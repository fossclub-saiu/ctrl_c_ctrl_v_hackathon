## 2. Demo Video (Maximum 5 minutes)

!!! info "Video Submission Process"
    1. Record your 5-minute demo video
    2. Upload to **Google Drive**
    3. Share access with `fossclub@saiuniversity.edu.in`
    4. Submit the Drive link via the Google Forms submission form

Your video must include the following sections:

### Problem Overview (30 seconds)
- What problem are you solving?
- Who is affected by this problem?

### Solution Demo (2-3 minutes)
- Live demonstration of your application
- Show all major features working
- Demonstrate the user flow

### Technical Explanation (1-2 minutes)
- Key features implemented
- Technologies used
- Challenges faced and how you solved them
- Architecture/logic decisions

!!! tip "Video Tips"
    - Keep it concise and focused
    - Avoid long introductions
    - Show actual working demo, not slides
    - Don't read from a script robotically
    - Ensure good audio quality - use a quiet environment
